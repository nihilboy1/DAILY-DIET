<h1>👾 | Execute na sua máquina</h1>
<ul>
    <li>
    1 - Realize o download do repositório
    </li>
    <li>
    2 - Abra o terminal na pasta raiz e execute `npm install` seguido de `npm run start`
    </li>
  <li>
    3 - Abra seu app `EXPO` e leia o `QRCODE` que vai aparecer
  </li>
</ul>
